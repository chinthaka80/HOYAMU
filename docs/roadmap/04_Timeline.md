## 4.2. Timeline Durations & Resource Allocation

- **Documentation (M1 - 2 Weeks):** Technical requirements validation, DB structural design, API specifications mapping.
- **UI/UX Design (M2 - 2 Weeks):** Figma layouts design and vector branding assets validation.
- **Backend API Core (M3 - 8 Weeks):** Sanctum auth pipelines, PostgreSQL migrations, categories lookup, notifications engines.
- **Frontend Next.js (M4 - 8 Weeks):** Interface components design, styling bindings, states handlers, search templates.
- **Marketplace Portals (M5 - 8 Weeks):** Jobs CV parser models, vehicle specs grid compare, real estate perch inputs.
- **Monetization & Vouchers (M6 - 4 Weeks):** LankaPay integration, Stripe secure callbacks, accounting transactions.
- **AI Integrations (M7 - 4 Weeks):** Google Vision APIs checks, DeepL multilingual translations workers.
- **QA & Testing (M8 - 4 Weeks):** PHPUnit database, JMeter load tests, manual visual regression scripts.
- **Deployment (2 Weeks):** Hetzner VPS environment setup, Docker Compose pipelines, Cloudflare DNS configurations.
- **Total Duration Estimate:** **36 – 48 Weeks** (9 – 12 Months).
