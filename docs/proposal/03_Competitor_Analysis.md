# Chapter 7: Competitor Analysis

## 7.1. Competitor Landscaping

HOYAMU.NET positions itself in the gaps left by existing local and global digital portals in Sri Lanka. Rather than competing directly in one narrow niche, we aggregate features to provide a unified experience.

| Platform | Core Strength | Key Vulnerabilities & Gaps | HOYAMU.NET Competitive Advantage |
| :--- | :--- | :--- | :--- |
| **Ikman.lk** | High volume of general classified listings. | Very high listing fees, outdated fuzzy search, zero reviews or merchant directory. | Freemium ad model + lower premium fees + instant fuzzy search. |
| **Daraz** | Established B2C product retail. | Limited strictly to physical items. Lacks Jobs, Properties, Vehicles, and local services. | Multi-niche ecosystem aggregating services and B2C merchants under one map layout. |
| **PickMe** | Local transport and food delivery. | Transactional transport focus. No classified listings, job boards, or directory reviews. | Display transit estimates on directory pages. B2B directory maps integration. |
| **Google Maps** | Global mapping and location data. | Lacks classified ads, local job listings, and localized customer support for SMEs. | Integrate location map pins directly within native job and property listing detail pages. |
| **Facebook Marketplace** | Large social user base, free posting. | Extremely low listing verification, high scam rates, and chaotic filtering. | User verification seals, AI fraud detection flags, and structured category templates. |

## 7.3. Core Advantage Pillars

1. **Typo-Tolerant Search (Meilisearch):** Allows users to find listings despite language typos (e.g. "Tooyota Acua" correctly matches "Toyota Aqua"), which is a major bottleneck on legacy local platforms.
2. **AI Translation Worker:** Automatic translation across English, Sinhala, and Tamil creates a 3x larger audience reach for every published ad.
3. **Freemium SME Directory:** Offers a completely free basic web landing page to local merchants, driving organic directory data accumulation ahead of monetized conversions.
