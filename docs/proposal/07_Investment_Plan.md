# Chapter 18: Investment Plan

HOYAMU.NET is raising **LKR 25,000,000** in Seed equity funding. This document details the allocation and targeted metrics of the investment capital.

---

## 18.1. Use of Funds Allocation

```
   [ LKR 25,000,000 CAPITAL INJECTION ]
                  |
     +------------+------------+
     |                         |
     v                         v
[ CAPEX (Infrastructure) ]  [ OPEX (Growth & Operation) ]
- Product Engineering (32%)  - Marketing & Sales (26%)
- Cloud Infrastructure (14%) - Salaries & Support (20%)
- Legal & Compliance (4%)    - Contingency Reserve (4%)
```

---

## 18.2. Detailed Capital Expenditure Breakdown

### 1. Product Engineering (32% - LKR 8,000,000)
- Core Figma UI/UX design system templates.
- Development of the Next.js frontend pages.
- Backend API services creation on Laravel 12.
- Mobile App creation using React Native.

### 2. Cloud Infrastructure (14% - LKR 3,500,000)
- Hetzner Cloud clusters configuration.
- PostgreSQL replica databases and Redis servers.
- Meilisearch instant search hosting nodes.
- Cloudflare R2 backup storage fees for Year 1.

### 3. Marketing & Merchant Acquisition (26% - LKR 6,500,000)
- Onboarding commissions for local sales agents (onboarding local merchants onto the directory).
- targeted social media campaigns (Facebook, YouTube, TikTok).
- Print marketing collaterals (business decals, QR code stickers).

### 4. Operational Salaries (20% - LKR 5,000,000)
- Compensation for critical engineering staff (CTO, systems developers).
- Salaries for helpdesk and listing verification moderators for 18 months.

### 5. Legal, Licensing & Reserves (8% - LKR 2,000,000)
- Corporate registry setup.
- Payment portal gateways integration (Stripe, LankaPay).
- Backup contingency fund for scaling spikes.
