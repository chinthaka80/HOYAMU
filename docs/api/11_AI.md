# API Chapter 11: AI Services API

These endpoints integrate with LLMs and translation APIs to provide smart assistance features.

---

## 11.1. Auto-Generate Ad Description (AI දැන්වීම් විස්තර සකස් කිරීම)
* **Endpoint:** `POST /api/v1/ai/generate-description` (Authorization required)
* **Request Body:**
```json
{
  "title": "Toyota Prius 2018 for Sale",
  "category": "vehicles",
  "key_attributes": {
    "condition": "Excellent",
    "mileage": "45,000 km",
    "options": "Leather seats, Sunroof"
  }
}
```
* **Response (200 OK):**
```json
{
  "success": true,
  "generated_description": "Selling a home used Toyota Prius 2018 in Kurunegala. Excellent condition car with a genuine mileage of 45,000 km. Comes with factory leather seats, a sunroof, and premium alloys..."
}
```

## 11.2. Translate Listing (දැන්වීම් පරිවර්තනය කිරීම)
* **Endpoint:** `POST /api/v1/ai/translate` (Authorization required)
* **Request Body:**
```json
{
  "text": "Excellent condition vehicle.",
  "target_languages": ["si", "ta"]
}
```
* **Response (200 OK):**
```json
{
  "success": true,
  "translations": {
    "si": "ඉතා හොඳ ධාවන තත්ත්වයේ පවතින වාහනයක්.",
    "ta": "சிறந்த நிலையில் உள்ள வாகனம்."
  }
}
```
