# Security Policy

## Supported Versions

Only the latest stable release versions receive security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not report it publicly. Email us directly at [security@hoyamu.net](mailto:security@hoyamu.net). We aim to respond within 48 hours and patch vulnerabilities within 14 days.
