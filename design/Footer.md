# Footer Layout Specifications

This document defines the layout structure and links grid for the platform's global footer.

---

## 1. Footer Grid Structure (Desktop)

The footer is divided into a 4-column layout spanning the full container width:

```
+-----------------------------------------------------------------------------------+
|  Column 1          Column 2          Column 3          Column 4                   |
|  [ Brand Info ]    [ Marketplace ]   [ Quick Links ]   [ Contact & Socials ]      |
|  Logo description  - Jobs            - About Us        - Email info               |
|  Copyright logs    - Vehicles        - Terms of Use    - Social icon grids        |
|                    - Properties      - Privacy Policy                             |
+-----------------------------------------------------------------------------------+
```

- **Column 1 (Brand Info):** Displays the official HOYAMU.NET logo, a brief ecosystem tagline description, and trademark copyright texts.
- **Column 2 (Marketplace):** Navigation links directing users to search lists filtered by category (Jobs, Vehicles, Properties, Directory).
- **Column 3 (Quick Links):** Links to legal terms, privacy policies, safety advice, and site maps.
- **Column 4 (Contact & Support):** Displays active support emails, phone numbers, and LankaPay partner badges.

---

## 2. Responsive Behavior

- **Mobile Viewports (< 768px):** Columns collapse into a stacked layout. Secondary link groups convert into accordion collapsible tabs to save screen space.
- **Background Color:** Structured Slate Dark (`#222222`) or deep charcoal backgrounds with white text links.
- **Typography size:** Muted description texts are styled at **12px** size, and header category columns use **14px bold** fonts.
