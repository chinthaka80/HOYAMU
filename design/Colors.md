# Color Palette Specifications

This document defines the official visual color language of HOYAMU.net.

---

## 1. Primary Colors

- **Primary Orange:** `#E25704` (Used for call-to-actions, primary highlights).
- **Hover Orange:** `#C94D03` (Used for CTA hover state transitions).
- **Premium Gold:** `#D4AF37` (Used for gold membership listings, verified seals, sponsored items).
- **Gold Light:** `#F0C75E` (Used for badges and status frames).

---

## 2. Interface Backgrounds & Text Colors

- **White:** `#FFFFFF` (Used for listing cards, modal overlays, main grids panels).
- **Background:** `#FAFAFA` (Used as global body viewport background).
- **Silver:** `#F2F2F2` (Used for alternating grid headers, code code blocks).
- **Dark:** `#222222` (Used for primary body text, headers, subheadings).
- **Gray:** `#6B7280` (Used for muted helper texts, dates, descriptions).

---

## 3. Feedback Indicators

- **Success:** `#16A34A` (Used for positive statuses, verification confirmations, pricing).
- **Danger:** `#DC2626` (Used for error messages, delete operations, flags).
- **Info:** `#2563EB` (Used for informational links, guides annotations).

---

## 4. Application Rules
1. **CTA Rule:** **Primary Orange** (`#E25704`) must be used for buttons representing main conversions (e.g., "Post Ad", "Apply Now").
2. **Premium Highlight:** **Gold** (`#D4AF37`) represents sponsored and verified listings.
3. **Contrast Standard:** Ensure all dark text on white backgrounds satisfies the WCAG 4.5:1 contrast standards.
